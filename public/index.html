Wordstock

Wordstock API

A spelling checking service
This service accepts a word and returns a response indicating
whether the word is spelled correctly, with suggestions if it is mis-spelled.

Documentation Summary

Resource Description
GET spell Returns a hash with the original word, a flag indicating if the word is spelled correctly, and an optional array of suggestions if the word is mis-spelled.

GET spell

Resource URL

http://wordstock.herokuapp.com/spell.json

Parameters

q (required) A single token representing a word. Exceptions are raised for null, blank or multiple words.

Example Request

GET 
http://wordstock.herokuapp.com/spell.json?q=scintillatingZ
 
{
  "original": "scintillatingZ",
  "correct": false,
  "suggestions": [
    "scintillating",
    "scintillation's",
    "scintillation",
    "scintillates",
    "scintillate"]
}
